# django_server
웹페이지를 제작하면서 마주하는 어려움을 해결 및 적용한 웹 프레임워크

---
## 📦 요구 사항
- Python 3.13+
- pip


## 🛠 설치
```bash
# 1) 저장소 클론
git clone https://github.com/jeonghyeon0329/django_server.git

# 2) 가상환경 & 패키지 설치 (pip)
python -m venv .venv
source .venv/bin/activate     # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```
